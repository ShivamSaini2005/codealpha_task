# 🛒 CodeAlpha_EcommerceStore

A Full Stack E-Commerce Store developed as part of the **CodeAlpha Full Stack Development Internship (Task 1)**.

## Features

- User Registration & Login
- Product Listing
- Product Details Page
- Shopping Cart
- Checkout & Order Processing
- Order History
- Responsive User Interface
- Express.js Backend
- JSON-based Database

## Tech Stack

### Frontend
- React
- TypeScript
- HTML5
- CSS3
- Vite

### Backend
- Node.js
- Express.js

### Database
- JSON (db.json)

## Installation

```bash
npm install
npm run dev
```

To start the backend:

```bash
npm run server
```

## Internship Task

This project fulfills **CodeAlpha Full Stack Development Internship - Task 1 (Simple E-Commerce Store)**.

Implemented requirements:

- Product Listing
- Product Details
- Shopping Cart
- User Registration/Login
- Order Processing
- Backend using Express.js
- Database for Products, Users and Orders
